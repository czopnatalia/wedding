# Wedding Website & Guest Management System 

A bespoke, fully responsive web application designed to serve as a digital invitation, RSVP platform, and collaborative photo hub for wedding guests.

## Features

* **Interactive Digital Invitation:** A sophisticated landing page with a real-time countdown to the ceremony, protected by a secure access gate.
* **Integrated RSVP System:** A seamless way for guests to confirm attendance, with data stored directly in the host's database.
* **Collaborative Photo Hub:** 
    * **Upload:** Guests can share their memories directly from their phones.
    * **Download:** Access to a shared gallery where guests can download photos in high quality.
    * **Emergency Backup:** Integrated link to Google Drive to handle high-volume uploads and full-resolution storage.
* **Admin Dashboard:** A private management panel for the hosts to:
    * Monitor guest statistics and RSVP status.
    * Moderate uploaded photos.
    * Generate and download detailed attendance reports.
* **Mobile-First Design:** Fully responsive UI, optimized for seamless use on smartphones during the event.
* **QR Code Integration:** Designed to be accessed instantly by scanning a QR code printed on physical wedding invitations.

## Technologies Used

* **Backend:** PHP 8.2 (with PDO for secure database communication)
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Custom Glassmorphism, Flexbox, Grid), JavaScript (Vanilla JS)
* **Security:** Session-based authentication and password-protected entry.
* **Deployment:** Live production environment on a custom domain.
